# Project Plan: CPNS Productivity Web App

## 1. Goal
Build a web app for CPNS candidates: focus, tracking, community.

## 2. Milestones
- **Phase 1: MVP (Minimum Viable Product)**
    - Week 1: Setup repo, basic UI, user auth.
    - Week 2: Pomodoro timer, basic task list.
    - Week 3: Simple progress tracking (completed tasks).
    - Week 4: User testing, bug fixing.
- **Phase 2: Enhancements**
    - Month 2: Community forum, advanced analytics, notification system.
    - Month 3: AI study plan (basic), additional habit trackers.

## 7. Risks

- User adoption → market via CPNS communities.
