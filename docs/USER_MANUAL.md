# User Manual: CPNS Productivity Web App (MVP)

Selamat datang di **CPNS Productivity Web App**! Aplikasi ini dirancang untuk membantu Anda tetap fokus, teratur, dan melacak progres belajar Anda untuk persiapan CPNS.

## 1. Memulai

### 1.1 Registrasi & Login
1.  Buka aplikasi di browser Anda.
2.  **Untuk Registrasi:** Klik "Daftar" atau "Register". Masukkan email dan password Anda, lalu klik "Daftar". Anda juga bisa memilih "Daftar dengan Google".
3.  **Untuk Login:** Jika sudah punya akun, klik "Masuk" atau "Login". Masukkan email dan password Anda, atau pilih "Masuk dengan Google".

## 2. Fitur Utama

### 2.1 Pomodoro Timer (Fokus Belajar)
Fitur ini membantu Anda belajar dengan teknik Pomodoro, yaitu interval fokus diikuti istirahat singkat.
1.  **Mulai Sesi:** Di dashboard utama, Anda akan melihat timer. Klik tombol "Mulai" atau "Start".
    -   Timer akan mulai menghitung mundur untuk sesi kerja (default 25 menit).
2.  **Istirahat:** Setelah sesi kerja selesai, Anda akan menerima notifikasi, dan timer akan beralih ke sesi istirahat singkat (default 5 menit).
3.  **Jeda/Lanjutkan:** Klik tombol "Jeda" atau "Pause" untuk menghentikan timer sementara. Klik lagi untuk "Lanjutkan" atau "Resume".
4.  **Reset:** Klik tombol "Reset" untuk mengulang timer dari awal.

### 2.2 Manajemen Tugas (To-Do List)
Atur semua tugas belajar Anda di sini.
1.  **Tambah Tugas Baru:**
    -   Di bagian "Tugas Anda" atau "Your Tasks", ketik tugas di kolom input.
    -   Tekan Enter atau klik tombol "Tambah Tugas" atau "Add Task".
2.  **Edit Tugas:**
    -   Klik ikon pensil (Edit) di samping tugas yang ingin Anda ubah.
    -   Edit teks, lalu tekan Enter atau klik ikon simpan.
3.  **Tandai Selesai:**
    -   Klik kotak centang di samping tugas untuk menandainya sebagai "Selesai".
    -   Tugas akan berpindah ke bagian "Tugas Selesai" atau "Completed Tasks" dan jumlah tugas selesai Anda akan bertambah.
4.  **Hapus Tugas:**
    -   Klik ikon tempat sampah (Delete) di samping tugas untuk menghapusnya secara permanen.

### 2.3 Pelacakan Progres (Progress Tracker)
Lihat seberapa banyak yang sudah Anda capai.
-   Di dashboard, Anda akan melihat:
    -   **Total Sesi Pomodoro Selesai:** Berapa kali Anda menyelesaikan sesi Pomodoro.
    -   **Total Tugas Selesai:** Berapa banyak tugas yang sudah Anda tandai selesai.
