# Product Requirements: CPNS Productivity Web App (MVP)

## 1. Overview
Web app to help CPNS candidates manage study time, track progress, and stay motivated, *beyond* just drill questions.

## 2. Core Features (MVP)

### 2.1 User Authentication
-   Register/Login via Email/Password & Google OAuth.
-   Secure session management.

### 2.2 Pomodoro Timer
-   Set custom work/break intervals.
-   Start/Pause/Reset timer.
-   Notification on session end.

### 2.3 Task Management
-   Add/Edit/Delete study tasks.
-   Mark tasks as completed.
-   Prioritize tasks.

### 2.4 Basic Progress Tracking
-   Display count of completed tasks.
-   Show total Pomodoro sessions completed.

## 3. Non-Functional Requirements
-   **Performance:** Fast load times (<3s).
-   **Scalability:** Supabase handles initial user growth.
-   **Security:** OWASP Top 10 mitigated.
-   **Usability:** Intuitive UI/UX for all tech levels.
-   **Responsiveness:** Mobile-first design.

## 4. Out of Scope (MVP)
-   Community features (forum, chat).
-   Advanced analytics (graphs, historical data).
-   AI-powered study plans.
-   Exam drill questions.
-   Push notifications (beyond browser).

## 5. Success Metrics
-   User retention (7-day, 30-day).
-   Average Pomodoro sessions per user/day.
-   Task completion rate.
-   Positive feedback in user testing.
